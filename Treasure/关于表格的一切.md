修改表格内间距

```tex
\setlength{\tabcolsep}{6pt} % General space between cols (6pt standard)
\renewcommand{\arraystretch}{1} % General space between rows (1 standard)
```





### 一些疑难



> \cline + \cellcolor  水平线会被遮挡

使用 `\hhline`

```tex
\usepackage{hhline}

\hhline{|~|-|}
% ~ 是 那一格不画线，- 是 那一格画线，需要几格就加几个
```
