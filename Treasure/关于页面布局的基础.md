### 页眉页脚

```
\pagestyle{type}`，`\thispagestyle{type}`，`setcounter{page}{number}
```

```
\pagestyle{type}`对后续所有页面都有效，而`\thispagestyle{type}仅仅对当前页面有效
```

- empty - 没有页眉和页脚
- plain - 没有页眉，页脚包含一个居中的页码
- headings - 没有页脚，页眉包含章/节或者字节的名字和页码
- myheadings - 没有页脚，页眉包含有页码
